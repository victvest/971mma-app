# App fonts

| Role | Family | Source | License |
|------|--------|--------|---------|
| Display (headlines) | General Sans | [Fontshare](https://www.fontshare.com/fonts/general-sans) | Free for commercial use (ITF EULA) |
| Body (UI copy) | Inter | `@expo-google-fonts/inter` | SIL Open Font License |

To swap fonts app-wide, edit `src/shared/theme/fonts.ts` (`fontStacks`) and update `src/core/fonts/useAppFonts.ts` asset map.
